# 🎨 Template de Prompts para Gemini - LANA Celo

## 📋 Prompt Inicial Completo

```
Diseña una landing page moderna y profesional para LANA Celo, una plataforma de 
inclusión financiera para América Latina construida en Celo blockchain.

CONTEXTO DEL PROYECTO:
- Nombre: LANA Celo
- Propósito: Remesas, servicios de pago y microfinanzas
- Blockchain: Celo
- Audiencia: Usuarios en América Latina
- Estilo: Moderno, profesional, confiable, inclusivo

REQUISITOS TÉCNICOS:
- Framework: Next.js 14 con App Router
- Estilos: Tailwind CSS
- Enfoque: Mobile-first, responsive
- Performance: Optimizado para carga rápida

PALETA DE COLORES BASE:
- Fondo oscuro: #0A1628 (azul oscuro)
- Fondo alternativo: #1a2332
- Verde Celo: #35D07F (acento principal)
- Texto principal: #FFFFFF (blanco)
- Texto secundario: #E5E7EB (gris claro)

ELEMENTOS PRINCIPALES:

1. LOGO:
   - Diseño moderno para "LANA Celo"
   - NO usar rayo/lightning bolt (era del proyecto anterior)
   - Representar: crecimiento, inclusión financiera, LATAM
   - Colores: Verde Celo como principal
   - Debe funcionar en: icono solo, texto solo, completo

2. HERO SECTION:
   - Fondo: Oscuro con patrón sutil (hexagonal o similar)
   - Tagline: "¡Envía, Paga, Crece!" - DEBE SER CLARAMENTE VISIBLE
   - Descripción: "Remesas, Servicios, Crédito. Comisiones Ultra Bajos."
   - Botón CTA: "Conectar Wallet" (verde Celo)
   - Badge: "Potenciado por Celo"
   - Layout: Centrado vertical y horizontalmente

3. SECCIONES ADICIONALES:
   - Features (6 características)
   - How It Works (3 pasos)
   - Benefits (estadísticas)
   - CTA Section
   - Footer

PROPORCIONA EN TU RESPUESTA:

1. DISEÑO VISUAL:
   - Descripción detallada del diseño
   - Layout de cada sección
   - Jerarquía visual

2. ESPECIFICACIONES TÉCNICAS:
   - Códigos de color exactos (hex)
   - Tamaños de fuente (px o rem)
   - Espaciado (padding, margin)
   - Breakpoints responsive

3. CÓDIGO:
   - SVG path para el logo (si aplica)
   - Clases de Tailwind CSS para cada elemento
   - Estilos CSS personalizados si es necesario

4. TIPOGRAFÍA:
   - Fuente recomendada (Google Fonts preferible)
   - Tamaños para h1, h2, h3, p
   - Pesos de fuente

5. EFECTOS VISUALES:
   - Sombras (text-shadow, box-shadow)
   - Gradientes (dirección, stops)
   - Animaciones (si aplica)
   - Hover effects

REQUISITOS ESPECÍFICOS:
- El texto "¡Envía, Paga, Crece!" DEBE tener excelente contraste
- El logo debe ser único y memorable
- Diseño debe ser moderno pero profesional
- Colores deben ser accesibles (WCAG AA mínimo)
- Todo debe ser responsive
```

## 💡 Tips para Usar con Gemini

1. **Sé Específico:**
   - Menciona problemas concretos ("texto borroso", "logo no me gusta")
   - Solicita soluciones específicas

2. **Pide Código:**
   - Siempre pide código Tailwind CSS
   - Pide SVG paths para logos
   - Pide estilos CSS si es necesario

3. **Itera:**
   - Empieza con diseño general
   - Luego pide ajustes específicos
   - Refina hasta que esté perfecto

4. **Documenta:**
   - Guarda todas las especificaciones
   - Guarda los prompts que funcionan
   - Reutiliza para futuros proyectos
